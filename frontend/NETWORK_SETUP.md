# Paseo Testnet Setup

## Network Details
- **Network Name**: Paseo Testnet (Polkadot Hub)
- **Chain ID**: 420420417
- **RPC URL**: `https://eth-rpc-testnet.polkadot.io/`
- **Currency Symbol**: PAS
- **Block Explorer**: `https://paseo.subscan.io`

## Add to MetaMask

### Option 1: Manual Setup
1. Open MetaMask
2. Click network dropdown → "Add Network"
3. Click "Add a network manually"
4. Enter the details above
5. Click "Save"

### Option 2: Programmatic (Add Network Button)
The app will prompt you to add the network automatically when you connect.

## Troubleshooting

### "Wrong network" error
- Switch to Paseo testnet in MetaMask
- Refresh the page

### "Could not decode result data"
- You're on the wrong network
- The contract doesn't exist on the current network
- Switch to Paseo testnet

### Get Test Tokens
Contact the Paseo testnet faucet or team for test PAS tokens.
